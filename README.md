🌐 Network Tools

A modern, all-in-one web-based networking toolkit designed for developers, cybersecurity learners, and IT enthusiasts.
Analyze, test, and explore network data directly from your browser — fast, clean, and powerful.

🚀 Features
🔍 Core Tools
🌍 IP Lookup — Get IP location, ISP, timezone & map
🌐 DNS Lookup — Resolve records via Cloudflare DoH
🕵️ Whois / RDAP — Domain registration info
📡 Ping Tool — Measure HTTP response time
🛣️ Traceroute — Track network path (API-based)
🔓 Port Scanner — Scan open ports via public API
🛠 Utilities
🧮 Subnet Calculator
🔐 Hash Generator (SHA-1 / SHA-256)
🔗 URL Encode / Decode
💻 Base64 Encode / Decode
🎨 UI / UX
✨ Modern dashboard design
🌙 Dark / Light mode toggle
📱 Fully responsive (mobile + desktop)
⚡ Smooth animations
🔔 Beautiful alerts using SweetAlert2
